# Release Profile

Graphical Abstract Creator is an EatRice Lab work.


## Current capability set

- Graphical abstract creation in editable PowerPoint format.
- Native PowerPoint text boxes for visible text.
- Native PowerPoint vector shapes, connectors, arrows, tables, and editable charts for graphical elements.
- Structured content brief intake with sufficiency scoring.
- Compact generation confirmation card for strict publication outputs.
- International English, Simplified Chinese, Traditional Chinese, and bilingual English-Chinese modes.
- Chinese top-journal style with restrained terminology and concise labels.
- Palette presets with editable vector palette-strip preview.
- Layout engines for pipeline, center-core, before-after, multiscale, comparison, and data-model-decision compositions.
- Vector object library for AI models, materials, mechanics, civil structures, sensors, FEM results, rare-earth magnets, optimization loops, and multiscale mechanisms.
- Preflight checks, editable-vector validation, and publication quality audit.


## Current

- Added Composition and compactness standard for publication-grade graphical abstracts.
- Added frame hierarchy rules for large core frames, support frames, and micro labels.
- Added typography rules for title, module labels, bilingual text, and manuscript-scale legibility.
- Added review heuristics for reducing equal-card workflow layouts into compact manuscript figures.
- Added Q9 composition compactness gate.
