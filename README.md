# 图形摘要创建器

图形摘要创建器是 **EatRice Lab 作品**，用于创建论文图形摘要。工作流为：读取内容文本，使用压缩风格确认卡确认方向，生成完整预览提示词，创建图形摘要预览图，审查预览图，按通过的预览图重建可编辑 PPT，并校验 PPT 与预览图一致。

## 安装到 Codex

1. 准备 `skill.zip`。
2. 打开支持 Skills 的 Codex 或 ChatGPT 环境。
3. 进入 Skills 或 Manage skills。
4. 上传 `skill.zip`。
5. 启用 `graphical-abstract-creator`。
6. 在对话中输入图形摘要内容概要。

## 最低输入

必须提供图形摘要要展示的内容。内容越详细越好，建议包含研究对象、机制或流程、方法或模型、关键结果、应用场景和参考图文。

## 压缩风格确认卡

Skill 只确认四组设置：语言与学术语气、视觉结构与复杂度、配色与信息密度、源图处理与输出内容。用户可以直接回答“使用默认设置”。

## 硬性审查规则

- 主模块最多四个；超过四个必须合并。
- 预览图必须比简单流程图更丰富。
- 必须有一个主视觉核心。
- 必须体现机制、过程、转化、耦合或结果关系。
- 必须有明确的大图框/小图框层级。
- 信息密度必须达到选定档位的下限。
- 图面必须符合学术审美，避免卡通化、海报化和装饰化。

## 输出标准

最终交付为可编辑 `.pptx`。所有可见文字使用 PowerPoint 文本框，视觉元素优先使用 PowerPoint 原生矢量对象。PPT 必须与已确认的预览图在构图、模块数量、主次层级、颜色、箭头、标签和核心表达上保持一致。
