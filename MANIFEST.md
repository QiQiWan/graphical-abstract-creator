# Manifest

- `SKILL.md`: main skill instructions.
- `references/`: working standards for interaction, layout, palettes, vector icons, Chinese journal style, prompt confirmation, and quality review.
- `scripts/check_graphical_abstract_spec.py`: JSON spec, strict-mode, and content sufficiency checker.
- `scripts/build_graphical_abstract_pptx.py`: editable vector PPTX builder with layout engines and vector object DSL.
- `scripts/generate_palette_strips.py`: editable vector palette-strip preview generator.
- `scripts/validate_pptx_editability.py`: PowerPoint vector-purity validator.
- `scripts/audit_graphical_abstract_quality.py`: publication-grade audit script.
- `examples/`: English, Chinese top-journal, and palette-strip sample PPTX/spec files.
- `docs/`: multilingual user documentation, installation/update docs, usage examples, API spec, palette strip docs, and quality gates.
- `tests/`: smoke tests for examples and scripts.
