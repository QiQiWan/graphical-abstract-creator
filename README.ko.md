# Graphical Abstract Creator

Graphical Abstract Creator is an **EatRice Lab** skill for publication-grade graphical abstracts. The workflow is: read the content brief, confirm the style, build a full prompt, generate a preview image, review the preview, reconstruct an editable PPT, and validate preview–PPT consistency. Please see `README.md` or `README.en.md` for the full documentation.
