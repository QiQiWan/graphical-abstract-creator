# 图形摘要创建器

创建可编辑矢量 PowerPoint 图形摘要。最低输入要求：说明图形摘要大致要展示什么，越详细越好。流程不再询问目标期刊、期刊族、学科方向或研究对象，这些信息从用户提供的内容概要和上传材料中推断。

## 核心流程

1. 提供详细的图形摘要内容概要。
2. 选择输出语言和期刊风格。
3. 确认或修改自动建议的核心论断、版式、配色、矢量重绘策略和不支持内容排除项。
4. 需要选择颜色时，先查看配色彩条。
5. 生成 PPTX。
6. 运行 spec 检查、可编辑性验证和质量审查。

## 严格保证

- 所有文字为可编辑 PowerPoint 文本。
- 所有视觉元素尽量为 PowerPoint 原生矢量图元。
- 不插入栅格图片、截图、嵌入媒体、OLE 对象或外部链接素材。
- 支持中文顶级期刊和中英双语模式。
- 内置配色彩条预览：`examples/palette_strips.pptx`。

更多安装、使用示例、API 规范、配色彩条和质量门控见 `docs/`。
