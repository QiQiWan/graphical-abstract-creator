# Layout Patterns

Supported patterns:
- left-to-right: for workflows, limited to 3-4 modules;
- center-core: for one dominant innovation, usually 3 modules;
- before-after: for contrastive results, usually 2-3 modules;
- multiscale: for scale coupling, usually 3-4 modules;
- comparison: for baseline-versus-proposed stories, usually 2-4 modules.

If a layout naturally produces more than four modules, merge secondary modules into callouts or nested sub-elements.
