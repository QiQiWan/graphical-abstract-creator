# Prompt Construction Standard

Build one detailed master prompt for the preview image.

The prompt must contain these sections:

1. **Scientific content**: what the figure must show, preserving only supported claims.
2. **Module plan**: no more than four main modules, with each module assigned a role.
3. **Visual-complexity floor**: request one dominant core object, at least three distinct scientific object types, mechanism cues, result/evidence cues, and supporting context.
4. **Mechanism completeness**: describe the transformation, interaction, coupling, pathway, model operation, or cause-effect relation that should be visible.
5. **Large/small frame hierarchy**: specify which area is the dominant region and which areas are secondary.
6. **Information density**: specify compact, standard, or rich density and the visible text budget.
7. **Academic aesthetics**: request a restrained publication figure with clean alignment, balanced whitespace, precise typography, non-decorative colors, and no poster-like card grid.
8. **Exclusions**: avoid simple flowcharts, equal-sized box rows, excessive bullets, generic stock icons, unsupported mechanisms, unsupported numbers, decorative clutter, and five or more main modules.

The prompt should describe a visually complete graphical abstract preview, not a schematic placeholder.
