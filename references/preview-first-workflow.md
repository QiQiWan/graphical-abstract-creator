# Preview-First Workflow

1. Read the content brief.
2. Confirm the style.
3. Build the master preview prompt.
4. Generate a graphical-abstract preview image.
5. Review the preview for scientific correctness, visual richness, composition, density, and top-journal suitability.
6. Iterate on the preview until accepted.
7. Reconstruct the accepted preview as an editable PowerPoint file.
8. Validate preview–PPT consistency.
