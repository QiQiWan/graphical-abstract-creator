# Image and Vectorization Standard

## Principle

Source images may guide conceptual redraws, but the final graphical abstract must use editable PowerPoint vector elements.

## Acceptable conversions

- Raster workflow screenshot -> simplified editable workflow blocks.
- Microscopy image -> schematic microstructure object.
- Model architecture image -> editable node/layer graph.
- Experiment photo -> simplified instrument/sample geometry.
- Result plot -> editable vector chart based on verified values.

## Unacceptable conversions

- Embedding PNG/JPG images.
- Pasting screenshots.
- Flattening SVG/PDF into an image.
- Tracing decorative images without scientific simplification.
- Adding visual features not visible or described in the source.

## Accuracy rule

Scientific accuracy outranks visual similarity. Simplify rather than fabricate details.
