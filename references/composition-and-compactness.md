# Composition and Compactness

Use one dominant visual composition. Avoid equal-weight card grids. The image should look like a compact publication figure.

Rules:
- one primary visual core should dominate;
- no more than four main modules;
- supporting modules should be smaller or quieter;
- keep title and bottom key message compact;
- use whitespace as structure, not empty decoration;
- avoid five-card workflow posters.
