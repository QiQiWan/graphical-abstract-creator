# Chinese Top-Journal Graphical Abstract Style

## General style

Use concise, rigorous, evidence-based Chinese. Avoid slogans, marketing adjectives, exaggerated novelty language, and unsupported causal language.

## Preferred verbs by evidence level

- Use **构建、建立、提出** only when a method or framework is clearly described.
- Use **揭示、阐明** only when mechanisms are supported by data or analysis.
- Use **验证、证实** only when validation results are explicitly provided.
- Use **提高、降低、抑制、增强** only with verified direction and preferably verified magnitude.
- Use **用于、支撑、辅助** for application statements when deployment evidence is limited.

## Text length limits

- Main title: 18-28 Chinese characters preferred.
- Block label: 4-10 Chinese characters.
- Callout: 8-16 Chinese characters.
- Secondary note: no more than 24 Chinese characters.
- Avoid full-sentence paragraphs on the figure.

## Bilingual rules

For bilingual output, choose one primary language. Use the secondary language as compact labels, not duplicate paragraphs.

## Punctuation and typography

Use Chinese punctuation in Chinese text. Avoid mixing too many slashes, parentheses, and abbreviations. Define abbreviations once when necessary.

## Prohibited wording without evidence

Do not use: 首次、突破性、颠覆性、显著、极大、完全、精准、最优、国际领先, unless the user provides explicit evidence and requests that wording.
