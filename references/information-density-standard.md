# Information Density Standard

Use one of three density profiles while preserving the four-module cap.

## Compact
- 2-3 main modules.
- 3-4 object types.
- 1-2 labels per module.
- Best for Chinese top-journal or single-column-friendly output.

## Standard
- 3-4 main modules.
- 3-5 object types.
- 1-2 labels per module.
- Best for most graphical abstracts.

## Rich
- 4 main modules only; never exceed four.
- 4-6 object types.
- Controlled secondary callouts.
- Best for multiscale or coupled stories.

The preview fails if it is below the selected density floor or exceeds the complexity ceiling.
