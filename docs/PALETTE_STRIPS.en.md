# Palette Strip Preview

This skill includes editable vector color-strip previews for all built-in palettes. The preview is not a screenshot. It is a PowerPoint file made of native rectangles and editable labels.

Open:

```text
examples/palette_strips.pptx
```

Regenerate:

```bash
python scripts/generate_palette_strips.py examples/palette_strips.pptx
```

Each strip is ordered as background, neutral, primary, secondary, accent, line, and text. Use one palette consistently in a single graphical abstract.
