# Quality Gates

- Q0: content brief sufficiency.
- Q1: compact style card completed.
- Q2: story compressed to four modules or fewer.
- Q3: master preview prompt generated.
- Q4: preview image passes hard gates.
- Q5: exact-view slide created; it should look the same as the preview when opened or exported.
- Q6: editable reconstruction slide created; title, frames, arrows, formulas, metrics, legends, and major labels must be editable.
- Q7: render-level visual parity checked for the exact-view slide.
- Q8: editability and structural consistency checked for the editable reconstruction slide.
- Q9: typography, palette, composition, academic aesthetics, information density, and module complexity verified.
- Q10: delivery package complete, with clear explanation of exact-view slide, editable reconstruction slide, and local raster panels.

## Default package structure

1. Slide 1: exact-view slide for visual parity.
2. Slide 2: editable reconstruction slide for editing text, arrows, frames, formulas, metrics, and legends.
3. Optional Slide 3: edit map and assumptions.


## Visual parity and editable-layer gates

- Slide 1 must visually match the preview.
- Slide 2 must contain editable title, module titles, main arrows, frames, formulas, metrics, legends, and key explanatory text.
- Slide 2 may keep localized high-quality scientific image panels.
- Do not present a full-slide screenshot as a fully editable PPT.
