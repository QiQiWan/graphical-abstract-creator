# Creador de resúmenes gráficos - Usage examples

## Minimum request

Create an editable vector PPT graphical abstract. Content brief: show data acquisition, AI modeling, physics constraints, validation, and application. Language: English. Journal profile: international top-journal. Redraw any source images as editable vector shapes. Output: PPTX + quality report.

## Chinese top-journal request

生成中文顶刊风格的可编辑矢量 PPT 图形摘要。内容概要：展示材料数据输入、智能模型预测、物理约束、结果验证和工程应用。输出中文 PPTX + JSON spec + 质量报告。
